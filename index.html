Smart, simple, study ‚Äî Suzy

Suzy is an end-to-end educational web app built with ASP.NET Core Razor Pages and EF Core. It helps students manage notes, auto-generate flashcards using a GenAI (Gemini) integration, create subject-wise mock tests, and track progress using analytics.

This documentation summarizes the project structure, how Suzy works, installation, and the specific contributions by the team ‚Äî including the backend & GenAI work by Ajay.

Architecture & How it works

High-level components

  • Razor Pages ‚Äî UI layer (.cshtml + PageModel) for Notes, Flashcards, Mock tests, Auth, and Chat endpoints.
  • EF Core ‚Äî Data access (AppDbContext), models for Note, Tag, NoteTag, Flashcard, User, TestResult, SessionLog.
  • GenAI Service ‚Äî Server-side wrapper for Gemini: prompt building, chunking, response parsing, validation, and chat orchestration.
  • Storage ‚Äî local file storage (uploads/) or cloud blob; DB for metadata and logs.
  • Auth ‚Äî ASP.NET Identity for local accounts; scaffolded hooks for external providers.

Typical workflow (user)

  1. Sign in ‚Üí Upload a note (PDF/text).
  2. Tag the note (subject / topic) ‚Äî saved via Tag & NoteTag.
  3. Open the Flashcards page ‚Üí choose a document ‚Üí system chunks text and calls GenAI.
  4. GenAI returns Q/A pairs ‚Üí server validates ‚Üí saved as Flashcards.
  5. Create or take mock tests; results are saved per user.
  6. Open Chat with Suzy ‚Üí ask for progress (e.g., ‚ÄúHow am I improving?‚Äù) ‚Üí Chat service reads TestResult & SessionLog, computes trends, and replies with a concise summary and suggestions.

Prerequisites & Installation

Prerequisites

  • .NET 6.0 SDK or later

Installation

Clone the repository

git clone https://github.com/maihazem607/Suzy-Study-Partner.git
cd Suzy

Install dependencies

dotnet restore

Set up the database

dotnet ef database update

Configure AI Service

Add your Google Gemini API key to suzy-gemini-key.json
# or configure via the website settings page

Run the application

Access the application

Open your browser and navigate to http://localhost:5000

Flashcard generation (server-side pseudocode)

public async Task<List<Flashcard>> GenerateFlashcardsFromText(string text) {
  var chunks = ChunkText(text, maxChars: 3000);
  var prompt = BuildFlashcardPrompt(chunks);
  var response = await _geminiClient.GenerateAsync(prompt);
  var qaPairs = ParseAndValidate(response);
  SaveFlashcards(qaPairs, documentId);
  return qaPairs;
}
